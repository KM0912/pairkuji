# 要求内容

## 概要

統計タブを一新し、現在のセッション内統計に加えて、過去のセッションを跨いだ通算統計・推移グラフ・ペア相性・直接対決成績を表示する。

## 背景

現在の統計タブはセッション内の勝率・ペア頻度・対戦頻度のみを表示しており、過去のセッションデータ（`practiceSessions`テーブル）が活用されていない。過去データを集約した通算統計や推移グラフを表示することで、プレイヤーの成長可視化やライバル意識の醸成、ペア相性の発見につなげたい。

## 実装対象の機能

### 1. 通算統計ダッシュボード
- 全セッション（過去のアーカイブ + 現在のセッション）を集約した個人別通算勝率
- セッション参加回数・通算試合数の表示
- 勝率ランキング表示

### 2. 勝率推移グラフ
- セッションごとの勝率を折れ線グラフで表示
- プレイヤー別に推移を確認可能
- 直近の推移がわかるシンプルなグラフ

### 3. ペア相性ランキング
- ペア（2人の組み合わせ）ごとの通算勝率を算出
- 勝率が高いベストペアのランキング表示
- 一定試合数以上のペアのみを対象（信頼性確保）

### 4. 直接対決成績
- 特定のプレイヤー同士が対戦側で対峙した際の通算成績
- 天敵・カモランキング

## 受け入れ条件

### 通算統計ダッシュボード
- [ ] 全アーカイブセッション + 現在のセッションの勝敗を集約した通算勝率が表示される
- [ ] セッション参加回数・通算試合数が表示される
- [ ] 勝率降順でランキング表示される

### 勝率推移グラフ
- [ ] セッションごとの勝率が折れ線グラフで表示される
- [ ] 複数プレイヤーの推移を同時に確認できる
- [ ] グラフはCSS/SVGで描画（外部チャートライブラリ不要）

### ペア相性ランキング
- [ ] ペアごとの通算勝率が計算・表示される
- [ ] 3試合以上のペアのみ表示（最低試合数フィルタ）
- [ ] 勝率降順でランキング表示される

### 直接対決成績
- [ ] 2人が対戦側にいた場合の通算勝敗が表示される
- [ ] 天敵（最も負けている相手）・カモ（最も勝っている相手）がわかる

## 成功指標

- 過去のセッションデータが活用され、プレイヤーの成長が可視化される
- 統計画面がセッション未開始時でも過去データがあれば表示される

## スコープ外

以下はこのフェーズでは実装しません:

- セッション一覧画面（個々のセッション詳細の閲覧）
- セッションMVP表示
- キャリーポイント等の高度な指標
- エクスポート機能
